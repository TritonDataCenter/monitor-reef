# Conversion Plans

This entire directory is a place for LLMs to deposit intermediate plans for
big projects that require a lot of coordination across multiple context windows.

In particular our restify-conversion skill specifies use of this directory.

Artifacts here are only guaranteed valid as of the their commit date.
They provide some basic insights into how the LLM-generated code was built. 
They are retained as potential fodder for improving our skills and prompts in the future.
